## 上传照片

首次上传需要一个 GitHub Personal Access Token:

1. 去 https://github.com/settings/personal-access-tokens → **Generate new token** → **Fine-grained**
2. Repository access 选 **Only select repositories** → 只勾 `qiqi-timeline`
3. Repository permissions 里把 **Contents** 设为 **Read and write**
4. 生成后复制 token(`github_pat_...` 开头)
5. 打开网站,点右下角 `+`,粘贴 token 就可以开始上传

Token 仅保存在你自己的浏览器 localStorage,不会出现在仓库里。
